backend

Navigate to the backend folder in the terminal.
Install the required dependencies by the entering the command "npm install".
Run the server by typing the command "npm start".
It will be connected to DB.

frontend

Navigate to the backend folder in the terminal.
Install the required dependencies by the command "npm install".
Run ther code by typing the command "npm start".
The output will be displayed on the localhost.

webpage

Click the sign up button in the login page.
Create a user by entering the name, email and password and click sign up button.
Come back to the login page, enter the correct email and password.
If email is wrong it will alert "Email doesn't exist".
If password is wrong it will alert "Invalid Credentials".
If both are correct it will alert "Login Success"
